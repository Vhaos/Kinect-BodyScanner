README for Console Kinect

1. Use VS Express 2013 for Windows Desktop (not Windows)
2. Its a XAML/WPF Application
3. Use .NET Framework 4.5.1
4. Press Stop to start scanning
5. Scan is saved to Desktop
6. Use MeshLab to open the .ply file or use this website http://bl.ocks.org/phil-pedruco/9913243 to open the .txt file. It outputs both files.
